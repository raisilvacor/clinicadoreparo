# Imagens

Coloque suas imagens nesta pasta para que sejam exibidas no site.

## Imagens dos Dispositivos

- `android.jpg` - Imagem para o módulo Android
- `iphone.jpg` - Imagem para o módulo iPhone
- `mac.jpg` - Imagem para o módulo MAC

## Imagens do Laboratório

Você pode adicionar quantas imagens quiser do seu laboratório:
- `lab1.jpg` - Primeira imagem do laboratório
- `lab2.jpg` - Segunda imagem do laboratório
- `lab3.jpg` - Terceira imagem do laboratório
- `lab4.jpg` - Quarta imagem do laboratório
- `lab5.jpg` - Quinta imagem do laboratório
- `lab6.jpg` - Sexta imagem do laboratório

**Nota:** O site está configurado para exibir até 6 imagens do laboratório. Se você quiser adicionar mais, edite o arquivo `templates/index.html` e adicione mais elementos `.lab-image-card` na seção "Nosso Laboratório".

## Formato Recomendado

- **Dispositivos:** Imagens retangulares (proporção 16:9 ou 4:3), resolução mínima 800x600px
- **Laboratório:** Imagens retangulares ou quadradas, resolução mínima 600x600px
- **Formato:** JPG ou PNG
